Simple workflow

From installed to presentation-ready in minutes.

ArVisual is an OBS source filter. There is no separate application window, virtual camera chain or external processing route.

  1. 1
    Install into OBSThe Windows installer detects the OBS root and validates it before copying plugin files.
  2. 2
    Add the source filterOpen Source → Filters → Effect Filters → ArVisual.
  3. 3
    Choose a presetStart with Product Clean Pop, Beauty Bright or MasAri Dopamine.
  4. 4
    Set Instant AmountUse one master control to fit the look to the source and lighting.

FAQ

Useful details before installing.

Is ArVisual a standalone application?

No. It is a native OBS Studio source filter and appears inside the OBS Filters window.

Does it replace manual color grading?

It is designed for fast creator workflows. Advanced productions may still use manual grading, but ArVisual provides a controlled presentation-ready starting point.

Why does the installer need the OBS root folder?

OBS plugins contain a native library and matching data files. They must be copied into the corresponding plugin and data directories inside the active OBS installation.

What version is the project built against?

The current build specification pins OBS Studio 31.1.1. Compatibility should be validated through release testing and issue reports.
